# pyproject2
Pattern Generator and Number Analyzer
Overview

The Pattern Generator and Number Analyzer is a Python console application that demonstrates the use of loops, conditional statements, control statements, and menus. Users can generate star patterns, analyze numbers within a specified range, and explore basic Python programming concepts through an interactive interface.

Features
Displays a welcome message.
Generates a right-angled triangle star pattern.
Identifies even and odd numbers in a user-defined range.
Calculates the sum of numbers within the selected range.
Demonstrates the use of if, else, break, and pass statements.
Interactive menu-driven interface.
Displays a thank-you message before exiting.
Technologies Used
Python 3
Concepts Demonstrated
Variables and Data Types

Conditional Statements (if, elif, else)
Loops (for, while)
Nested Loops
Control Statements (break, pass)
Menu-Driven Programming
How to Run
Install Python 3 on your computer.
Save the program as main.py.
Open a terminal or command prompt.

Run the program:

python main.py
Follow the on-screen instructions.
